

## Dependency

### Base
Basic usage assumes that you are not using any features in the project.
```toml
[dependencies]
sorrow-i18n = "0.1.0"
```
### Features

#### incl_dir
A simple feature for loading static files into a project.  

Usage:
```toml
[dependencies]
sorrow-i18n =  { version = "0.1.0", features = ["include_dir"] }
```
